# Módulo de Tarjeta de Contenido Personalizada

Módulo personalizado de HubSpot para crear múltiples tarjetas de contenido con icono, encabezado y descripción.

## Características

- **Múltiples elementos**: Agrega tantas tarjetas como necesites
- **Icono flexible**: Soporta emojis, iconos FontAwesome o imágenes personalizadas
- **Encabezado**: Título personalizable para cada tarjeta
- **Descripción**: Texto rico (rich text) para descripciones detalladas
- **Personalización de estilos**: Colores y alineación configurables
- **Grid configurable**: Controla cuántas columnas mostrar en desktop (1-4) y mobile (1-2)
- **Diseño responsivo**: Optimizado para todos los dispositivos
- **Animaciones suaves**: Efectos de hover y transiciones escalonadas

## Campos Configurables

### Contenido (Grupo Repetible)
- **Tipo de Icono**: Emoji, FontAwesome o Imagen
- **Icono**: Dependiendo del tipo seleccionado
- **Encabezado**: Título de la tarjeta
- **Descripción**: Texto descriptivo (rich text)

### Estilos
- **Color de Fondo**: Color de fondo de las tarjetas
- **Color del Texto**: Color principal del texto
- **Color del Icono**: Color del icono (solo FontAwesome)
- **Color del Borde**: Color del borde de las tarjetas
- **Alineación**: Izquierda, Centro o Derecha
- **Columnas en Desktop**: 1, 2, 3 o 4 columnas (por defecto: 4)
- **Columnas en Mobile**: 1 o 2 columnas (por defecto: 1)

## Uso

1. Agrega el módulo a tu página en HubSpot
2. Haz clic en "Agregar elemento" para crear nuevas tarjetas
3. Para cada tarjeta:
   - Selecciona el tipo de icono que deseas usar
   - Configura el icono según el tipo seleccionado
   - Ingresa el encabezado y descripción
4. Personaliza los colores, alineación y número de columnas según tus preferencias

## Diseño Responsivo

- **Desktop**: 4 columnas por defecto (configurable: 1-4)
- **Tablet (≤1024px)**: 3 columnas automáticamente
- **Mobile (≤768px)**: 1 columna por defecto (configurable: 1-2)
- **Mobile pequeño (≤480px)**: 1 columna

## Ejemplos de Uso

- Tarjetas de características de producto
- Tarjetas de servicios
- Tarjetas de beneficios
- Tarjetas informativas
- Grid de contenido destacado
